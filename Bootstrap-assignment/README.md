# Bootstrap Assignment: Responsive Web Page

## Overview
This project demonstrates the use of **Bootstrap 5** to create a fully responsive web page.  
The assignment covers forms, tables, images, buttons, and navigation components.  
It was built as part of a web development learning assignment to practice Bootstrap concepts.

---

## Features

### 1. Responsive Navigation Bar
- Collapsible hamburger menu on smaller screens.
- Links to Home, About, and Contact sections.
- Styled with Bootstrap utilities.

### 2. Registration Form
- Fields: First Name, Last Name, Email, Password, and Terms & Conditions checkbox.
- First Name and Last Name positioned side by side using Bootstrap grid.
- Email and Password fields are required.
- Submit button styled with `btn-success`.

### 3. Table for Displaying Data
- Hard-coded sample user submissions.
- Striped and hoverable rows for readability.
- Responsive table that scrolls on small screens.

### 4. Image Section
- Large responsive image using `img-fluid`.
- Circular image using `rounded-circle`.
- Two buttons:
  - One always visible.
  - One hidden on small screens using responsive utilities (`d-none` and `d-md-inline-block`).

### 5. Styling
- Custom CSS for fonts, spacing, table alignment, and image borders.
- Bootstrap classes handle most of the layout and responsiveness.
